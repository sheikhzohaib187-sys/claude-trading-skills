# Claude Trading Skills

SMC-based trading skills for Claude AI — built for NAS100, Forex, and Prop Firm challenges.

## Skills Included

| Skill | Description |
|---|---|
| quant-trader | SMC + multi-timeframe analysis for NAS100/Forex |
| trading-signals | Multi-market technical analysis (Elliott Wave, Wyckoff, Fibonacci) |
| trading-signals-pro | Institutional-grade crypto signals with confidence intervals |
| trading-strategy | Strategy development and evaluation |
| trading-coach | Trade review and CSV analysis |
| trading-signal-strategy-implementation | Order book depth signal generation |

## Usage

Each skill folder contains a SKILL.md file. Load into Claude skills directory at /mnt/skills/user/

## Context

Built for Funded Hive Phase 1 challenge (8% target), NAS100 NY session, SMC methodology.
